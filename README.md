🧩 Présentation du projet
Ce projet analyse un dataset RH complet contenant des informations sur les employés. L’objectif est de construire un pipeline Data Analyst professionnel couvrant l’exploration, le nettoyage, la création de vues analytiques, la visualisation Power BI et la mise en place d'un site web interactif.

🏗️ Pipeline du projet
1. Exploration & Nettoyage (MySQL)
Analyse initiale du dataset pour détecter erreurs, incohérences, doublons et valeurs manquantes.
Correction des anomalies, harmonisation des formats, standardisation des catégories.
Renommage des colonnes en français pour améliorer la lisibilité et garantir la qualité des données.

2. Analyses & Modélisation SQL
Vérification de la structure et cohérence des données.
Analyses métier : effectifs, turnover, ancienneté, répartition par genre, postes, départements.
Création de SQL Views dédiées pour faciliter l’intégration dans Power BI.

3. Dashboard Power BI
Dashboard RH interactif intégrant plus de dix visualisations :
KPIs clés (effectif actif, turnover, âge moyen, ancienneté).
Répartition par département, genre, poste, site.


4. Site Web (KPIs + Visualisations)
Développement d’un site web présentant :

KPIs RH essentiels

Graphiques interactifs

Distributions et analyses

Résumé des insights clés
Objectif : maximiser l’impact auprès des recruteurs et démontrer une maîtrise complète du pipeline Data → BI → Web.
